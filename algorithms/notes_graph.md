## O(n)
    - O(1)
        - constant time operation such as adding element to the end of the list  

    - O(log(n))
        - binary search 

    - O(n)
        - linear time  

    - O(nlog(n))
        - looping via list and perform binary search 

    - O(n^2)
        - loop via list n times 

    - O(2^n)
        - loop via tree which has n depth
